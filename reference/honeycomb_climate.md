# Rainfall and Temperature in The Americas

Monthly temperature and rainfall between 2009 and 2019 in 72 hexagonal
cells covering The Americas.

## Usage

``` r
data(honeycomb_climate)
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with
9432 rows and 4 columns.
