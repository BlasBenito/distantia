# Hexagonal Grid

Sf data frame with hexagonal grid of the dataset
[honeycomb_climate](https://blasbenito.github.io/distantia/reference/honeycomb_climate.md).

## Usage

``` r
data(honeycomb_polygons)
```

## Format

An object of class `sf` (inherits from `data.frame`) with 72 rows and 2
columns.
