# Coordinates of 100 Major Cities

City coordinates and several environmental variables for the dataset
`cities_temperature`.

The full dataset with 100 cities can be downloaded from
<https://github.com/BlasBenito/distantia/blob/main/data_full/cities_coordinates.rda>
(use the "Download raw file" button).

## Usage

``` r
data(cities_coordinates)
```

## Format

sf data frame with 5 columns and 100 rows.
