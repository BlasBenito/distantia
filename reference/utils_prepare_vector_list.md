# Convert List of Vectors to List of Data Frames

Convert List of Vectors to List of Data Frames

## Usage

``` r
utils_prepare_vector_list(x = NULL)
```

## Arguments

- x:

  (required, list of vectors) Default: NULL

## Value

List of data frames
